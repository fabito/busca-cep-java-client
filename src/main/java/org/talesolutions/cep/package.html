Invoca a busca de CEP disponível no site dos correios e extrai os resultados do html retornado.

Esta implementação utiliza HtmlUnit para fazer a requisição GET ao serviço dos Correios e também fazer o extrair os dados do HTML via XPath. O serviço invocado por essa implementação roda na seguinte URL:
http://www.buscacep.correios.com.br/servicos/dnec/consultaLogradouroAction.do
